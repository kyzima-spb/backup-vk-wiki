# [Инструменты Front-end разработчика](https://vk.com/mosginfo_frontend_tools)

Последние несколько лет Frontend активно развивается, появляются все новые и новые инструменты, которые позволяют:
- автоматизировать повторяющиеся действия;
- использовать новые возможности, не смотря на слабую поддержку в браузерах;
- использовать расширенный синтаксис описания стилей (CSS);
- использовать шаблонизаторы в HTML и JavaScript;
- выполнять автоматическое тестирование;
- и это далеко не самый полный список.

Использование изученных инструментов, позволяет сократить время разработки, и увеличить эффективность вашей работы, что в сумме дает Вам преимущество в жесткой конкурентной борьбе.
